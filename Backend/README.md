# RESTApi in Golang

### To Run this project

```bash
  go run main.go
```

### To Run Test 

```bash
  go test
```

# All Routes

**GET Routes**
| Route | Description |
| --- | --- |
| `/users` | List all Users |

**POST Routes**
| Route | Description |
| --- | --- |
| `/user` | Add User |

**PUT Routes**
| Route | Description |
| --- | --- |
| `/user/:id` | Update specific User |

**DELETE Routes**
| Route | Description |
| --- | --- |
| `/deleteuser/:id` | Delete specific User |
| `/deletealluser` | Delete all Users |

## Tech Stack

**Language:** Golang
